掃描當前目錄下所有 LLM 訓練任務，顯示狀態和效能摘要。

## 掃描邏輯

1. 使用 Glob 找出所有 `*/task.yaml` 檔案
2. 讀取每個 task.yaml 取得任務資訊
3. 讀取 versions/ 目錄取得最新版本資訊
4. 整理成表格顯示

## 輸出格式

```
LLM 訓練任務總覽
================

┌─────────────────────┬────────────┬─────────┬───────────┬───────────────┐
│ 任務名稱            │ 狀態       │ 版本    │ Macro-F1  │ 更新時間      │
├─────────────────────┼────────────┼─────────┼───────────┼───────────────┤
│ {task_name}         │ {status}   │ {ver}   │ {score}   │ {updated}     │
└─────────────────────┴────────────┴─────────┴───────────┴───────────────┘

狀態說明:
- created: 已建立，尚未配置
- configuring: 配置中
- training: 訓練進行中
- evaluating: 評估中
- deployed: 已部署
- archived: 已封存
```

## 無任務時

如果沒有找到任何任務：

```
目前沒有任何訓練任務。

使用 /nlp-skills:new-task 建立新任務，
或使用 /nlp-skills:coach 開始教練式引導。
```

## 執行步驟

1. 搜尋 task.yaml 檔案
2. 解析每個任務的配置
3. 取得最新版本的評估結果
4. 格式化輸出表格
