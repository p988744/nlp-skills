## 任務特點

### 分類 vs 生成

| 特性 | 分類任務 | 生成任務 |
|------|----------|----------|
| 輸出形式 | 固定標籤 | 自由文本 |
| 評估指標 | F1, Accuracy | BLEU, ROUGE, 人工 |
| 訓練複雜度 | 低 | 中-高 |
| 資料需求 | 相對少 | 相對多 |
| 對齊需求 | 通常不需要 | 可能需要 |
